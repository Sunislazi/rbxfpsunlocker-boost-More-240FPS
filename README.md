## Overview
**RBXFPSUnlocker** is a lightweight and effective utility that removes Roblox's default 60 FPS cap, allowing you to achieve higher frame rates (up to your monitor's refresh rate) for smoother and more responsive gameplay.

> [!Important]
> This is a third-party tool. Use at your own risk. Roblox may update their client and temporarily break compatibility.

## Key Features
- Removes the 60/240 FPS limit in Roblox
- Supports high refresh rate monitors (144Hz, 240Hz, 360Hz+)
- Lightweight and low resource usage
- Automatic injection when Roblox launches
- Custom FPS limit option
- Portable version (no installation required)
